# React Hooks: Because Classes Were Making Everyone Cry

React Hooks arrived in 16.8 like a superhero bursting through a wall shouting,  
“STOP WRITING CLASS COMPONENTS, YOU FOOLS!”

Suddenly, state and side effects were available in functional components.  
Developers rejoiced.  
Documentation teams wept.

## The Essentials

- `useState`: Your new best mate.  
- `useEffect`: Your new worst enemy if you forget the dependency array.

## Why Hooks Matter

Hooks make your code cleaner, simpler, and easier to reuse.  
They also make it easier to accidentally create infinite loops that melt your CPU.

If you’re writing React in 2026 without Hooks, Scrimshaw will personally  
jump on you and restyle you hair.
