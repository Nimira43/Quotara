# The Note‑Taking App That Refuses to Behave

Every dev builds a note‑taking app at some point.  
But this one?  
This one has *attitude*.

**Tabula** is an upcoming project from a new indie dev who decided that  
note‑taking apps are too boring and need more chaos.

## Features

- Notes that auto‑sort themselves based on “vibes”  
- A search bar that occasionally offers unsolicited advice  
- A dark mode so dark it absorbs nearby light  
- A settings page that apologises for existing

## Early Impressions
